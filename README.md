# Dotfiles and helpers to bootstrap mac machines to my liking

## Bootstrap and initial setup

1. Install and verify Xcode's command lines developer tools

```bash
xcode-select --install
xcode-select -p
```

## Notes for a more organized `$ZDOTDIR`/`$ZSH`

- Use `$HOME/.zshenv` to set `PATH`.
Since PATH is bound to be specific to the local machine, this one stays out of git. It is generated by the `_bootstrap/init.sh` script.
- Other env vars that can't or for some reason shouldn't be shared should go in the same file. Sharea-ble env vars go in `$ZDOTDIR/.zshrc`
- Aliases go in `$ZSH_CUSTOM/aliases.zsh`, functions in `$ZSH_CUSTOM/functions.zsh`
- Executables go in `$ZSH_CUSTOM/bin`

## Components

### Nvim

#### Setup

- `ln -s <dotfiles>/nvim .config/nvim`

#### General dependencies

- `brew install cmake` (used with telescope-fzf-native)
- `brew install fd`

FZF itself is installed as a dependency of nvim-bqf.

#### LSP/formatter dependencies

- [typescript tools](https://github.com/typescript-language-server/typescript-language-server)
- [cssls, jsonls and eslint](https://github.com/hrsh7th/vscode-langservers-extracted)
- [rustaceanvim](https://rust-analyzer.github.io/)
- [lua_ls](https://github.com/luals/lua-language-server)

Other lang servers used, e.g. tailwind, may be only temporary and I will not list them here.
